Disable Node Menu Item

Installation
------------

Install as normal.  Once installed you can set the content types you would
like to use this functionality at: /admin/structure/menu/dmni

Note: Core patch required to clear  PHP warning
-----------------------------------------------
The PHP warning described in this issue will pop up after saving a node:
https://www.drupal.org/node/2775665
Applying the patch attached to that issue will resolve the PHP warning.
